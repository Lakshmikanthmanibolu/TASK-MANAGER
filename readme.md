Task Manager Web Application
A responsive Task Manager web application built with HTML, CSS, and JavaScript. This project helps users manage daily tasks efficiently with dynamic task updates, a clean interface, and browser-based storage.

Objective
Develop a Task Manager Web Application using HTML, CSS, and JavaScript to manage daily tasks efficiently.

Project Structure
text
Task-Manager/
├── index.html
├── style.css
├── script.js
└── README.md
Features
Add new tasks

Display task list dynamically

Mark tasks as completed

Delete tasks

Filter tasks by All, Pending, and Completed

Responsive and professional user interface

Instant DOM updates without page reload

Local storage support to preserve tasks after refresh

Quick-add task suggestions for better usability

Functional Expectations Covered
Clean and responsive user interface

Proper DOM manipulation using JavaScript

Tasks update dynamically without page reload

Separate files for HTML, CSS, and JavaScript

No inline CSS or JavaScript used

HTML5

How to Run
Create a project folder.

Save the source code into these files:

index.html

style.css

script.js

Place all files in the same folder.

Open index.html in any modern web browser.

How to Use
Enter a task in the input field.

Click the Add Task button or press Enter.

View the task in the task list.

Mark a task as completed using the checkbox or complete button.

Delete a task using the delete button.

Use filters to view all, pending, or completed tasks.
